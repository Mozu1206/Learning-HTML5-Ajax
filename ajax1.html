4. [基本] Ajaxとは何か

Ajax（Asynchronous JavaScript + XML、エイジャックス）については、Ajaxとは｜「分かりそう」で「分からない」でも「分かった」気になれるIT用語辞典を参考にするとわかりやすいので、まずは見てください。基本的には、上記のサイトにも書いてあるように、JavaScriptを使って非同期通信をするんだなあという風に理解できていれば問題ないです。
Ajaxは基本的に以下の技術を組み合わせて実装されます。

  • XMLHttpRequest
  • JavaScript
  • DOM
  • XML
XMLHttpRequest
API

XMLHttpRequestとは、クライアントとサーバーの間でデータを伝送するための機能をクライアント側で提供するAPIです。簡単にまとめると、ブラウザ上でサーバーとHTTP通信を行うためのAPIです。このAPIを使って実装を行なっていくのがJavaScriptです。


JavaScript
プログラミング言語

前述のXMLHttpRequestは、JavaScriptの組み込みオブジェクトです。したがって、JavaScriptを使わないと実装ができません。


DOM
API

DOM（Document Object Model）とは、XMLやHTMLの各要素、たとえば<p>とか<img>とかそういった類の要素にアクセスする仕組みのことです。このDOMを操作することによって、要素の値をダイレクトに操作できます。例えば<p>の中身のテキストを変更したり、<img src="sample.jpg">のsrcの中身を変更して別の画像に差し替えるといったことができます。

XML
プログラミング言語

書やデータの意味や構造を記述するためのマークアップ言語の一つ（HTMLと似たようなもの）です。XMLはタグを自由に設定でき、そのタグに意味づけをすることができます。複数のデータを同時にやりとりしても、どのデータがどの要素なのか一発で判明できるのが強みです。
